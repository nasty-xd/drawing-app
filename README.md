Drawing App
##Project Description

**Drawing App** is a simple and interactive web application for creating digital drawings. Users can choose colors, adjust brush sizes, draw on a canvas, and save their artwork. The app also features a gallery to display saved drawings.

Key features include:

Color Selection – choose from preset colors or use a custom color picker

Brush Size – adjustable brush size for precise drawing

Canvas Actions – clear the canvas, save drawings, or start a new one

Gallery – view saved drawings in a gallery section

Interactive UI – toolbar with intuitive controls and real-time updates

---

## Technologies

- **HTML5 & CSS3** – structure, styling, and layout

- **JavaScript** – core drawing logic and event handling

- ** Canvas API** – rendering drawings on the canvas

- ** Font Awesome** – icons for toolbar actions

- ** DOM Manipulation** – dynamic updates for brush size, color, and gallery

---

## Installation and Running

1. Clone the repository:

```bash
git clone <repository_URL>
cd ts_todo
```

2. Install dependencies:
npm install

3. Build the project:
npm run build

4. Open index.html in a browser.
